# nestjs-st

Monorepo-практикум: **NestJS** (API) + **TanStack Start** (SSR/SEO) + **Nx**. Подробный план шагов — [`docs/development-roadmap.md`](docs/development-roadmap.md).

## Требования

- Node.js >= 20.10

## Команды (корень репозитория)

```bash
npm install
npm run start:dev    # Nest в workspace app
npm run test
npm run test:e2e
npm run build
```

Документация и первый урок: [`docs/README.md`](docs/README.md).
